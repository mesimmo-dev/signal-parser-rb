# Signal Parser RB

A lightweight Ruby script for parsing opportunity records and ranking them by signal strength.

This project is intentionally small. It is designed as a simple systems-oriented artifact that demonstrates structured thinking, scoring logic, and basic Ruby scripting.

## Purpose

Signal Parser RB evaluates a small list of projects, leads, roles, or opportunities and produces a ranked summary based on weighted criteria.

It can be used to compare:

- project ideas
- job leads
- collaboration opportunities
- publishing targets
- platform signals
- lightweight strategic options

## Core Idea

Not every opportunity deserves the same amount of attention.

This script helps identify which records have the strongest signal based on practical factors such as alignment, leverage, credibility, and execution ease.
